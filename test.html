What is HTML?

HTML stands for HyperText Markup Language. It's the standard language used to create web pages. HTML uses special codes called tags to tell the browser how to display content.

Basic HTML Structure

Every HTML page has this basic structure:

  • <!DOCTYPE html> - Tells browser it's HTML5
  • <html> - Root element that contains everything
  • <head> - Contains page information (not visible)
  • <body> - Contains all visible content

Key Things to Remember

  • HTML tags are surrounded by angle brackets: < >
  • Most tags come in pairs: opening <tag> and closing </tag>
  • Some tags are self-closing: <img>, <br>
  • Proper indentation makes code easier to read
  • Always include alt text for images for accessibility
  • Test your HTML in a web browser to see how it looks
